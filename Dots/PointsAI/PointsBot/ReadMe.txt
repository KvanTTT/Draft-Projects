PointsGame readme
=================

Описание
--------------
PointsGame - достаточно широко известная игра "точки". Основная цель проекта - разработка искусственного интеллекта.
Для работы необходим ИИ в виде исполняемого файла (exe), работающего по протоколу Points AI Protocol 6, либо динамическая библиотека (dll). Исполняемый файл должен иметь имя PointsBot.exe, библиотека - PointsBot.dll. С каким типом ИИ работать - определяется в настройках игры.
Если не планируется использование библиотеки, или будет использоватся библиотека той же разрядности, что и Windows, можно использовать для игры PointsShell.exe. В противном случае разрядность PointsShell должна совпадать с разрядностью библиотеки.
Для русификации интерфейса необходимо положить папку ru-RU с находящимися в ней локализованными ресурсами рядом с исполняемым файлом PointsShell.
Для работы понадобятся:
1. DirectX (http://www.microsoft.com/downloads/ru-ru/details.aspx?FamilyID=2da43d38-db71-4c1b-bc6a-9b6652cd92a3&displayLang=ru).
2. .NET Framework 4.0 или старше (http://www.microsoft.com/net/).
3. Microsoft Visual C++ 2010 Redistributable Package (http://www.microsoft.com/downloads/ru-ru/details.aspx?FamilyID=a7b7a05e-6de6-4d3a-a423-37bf0912db84&displaylang=ru).

Настройки
--------------
Game preferences:
	Width - ширина игрового поля. Максимум 39.
	Height - высота игрового поля. Максимум 32.
	Game type:
		Standart - классические правила игры.
		Always - базы обводятся всегда (включая пустые).
		Always enemy - поставленную в пустую базу точку всегда захватывает противник.
	Begin pattern:
		Clean - начало с чистого поля.
		Crosswire - начало со скреста.
		Square - начало с квадрата.
AI preferences:
	AI - включить искусственный интеллект.
	AI type:
		DLL - использовать динамическую библиотеку (dll).
		Console - использовать исполняемый файл (exe) по протоколу Points AI Protocol 6.
	Get move method:
		Simple - простой способ получения хода.
		With complexity - для получения хода будет использоваться указанная сила.
		With time - для получения хода будет использоваться указанное время.
	Complexity - сила игры ИИ. Изменяется от 0 до 100.
	Time - время на ход ИИ.
Players preferences:
	Red name - имя 1-ого игрока.
	Black name - имя 2-ого игрока.
	Red color - цвет 1-ого игрока.
	Black color - цвет 2-ого игрока.
Other preferences:
	Tab name - название вкладки с игрой.
	Filling alpha - прозрачность заливки.
	Background color - цвет фона.
	Full fill - заливка по максимальной площади.
	Sounds - звуки ходов.
